  • P1013LowerBound
  • Description

    • You are given a sequence A[1], A[2], ..., A[N] . ( |A[i]|<=2*10^9, 1~100000 ).(L,R,V) : find the smallest number of A[i] such that L<=i<=R and A[i]>V, if not exist, output not exist.\nGiven M queries, your program must output the results of these queries.

  • Input

    • The first line contains a single integer T, the number of test cases.\nFor each case, there are two integers N and M (1<= N, M <=100000).\nThe next line contain N elements.\nA1 A2 AN\nThe next M lines contain the operation in following form.\nL1 R1 V1\nL2 R2 V2\n��\nLM RM VM

  • Output

    • For each question, output the answer in one line.

  • Sample Input 1
    •         1
        5 5
        5 4 3 2 1
        1 5 2
        2 4 0
        3 5 3
        1 4 3
        2 5 1
  • Sample Output 1
    •         3
        2
        not exist
        4
        2
