  • XOR of bit operation

    • What is XOR?
    The operation of XOR will compare the same bit from two numbers,
    if the bit is different, there will be 1,else there will be 0.
    e.g:
    3 = 11(B)
    2 = 10(B)
    3 ^ 2 = 11(B) ^ 10(B) = 01(B) = 1
    There are three laws:
    (1)Commutative law: a^b^c = a^c^b
    (2)Any number XOR = itself: n^0 = n
    (3)Two same ame numbers XOR each them = 0: n^n = 0

  • Copying list in python

    • In a function, if assigning a value to a list which as an argument will result in a new copy.
    e.g:

    def function(nums):
    tmp = [1,2]
    print(id(tmp))
    nums = tmp[:]
    print(id(nums))
    nums[:] = tmp[:]
    print(id(nums))

    The first id is a1, the second id is a2 and the third id is a1.
    Therefore, if we want to give a value to nums,
    we should use nums[:], not num.

  • Use of XOR

    • There is a example about XOR:
    In an array, only one number appears once,and the rest two times.
    Find out the number.
    We can use XOR, the question will be very simple.
    Assumed the number of k just appeared once and the array has n numbers.
    Then a1^a2^a3...^an = k
    Because same numbers will be 0 and finilly the formula will be 0^k = k
    Code:

    int FindOnceNumber (int a[])
{
    int res = 0;
    for (int x:a)
        res = res ^ x;
    return res;
}
